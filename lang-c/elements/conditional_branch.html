条件分岐

条件分岐とは，条件に応じて行う処理を変化させるようなプログラムの構造である。 プログラムに柔軟さをもたせるために必須であり，非常に重要である。

if 文

if 文は，ある条件を満たしているときに，特定の処理を実行するような 選択的な処理をするために使われる。

ただし，if 文により実行されるのは続く１つの文だけであるため，
のように書くと，a > 100 でないとき改行文字だけ出力されることになる。 プログラムに処理を追加することはよくある。 そのとき，このような書き方は意図しない条件から外れた処理を誘発するため，
のように単純文で済むものをはじめから複合文として書くことを推奨することが多い。 そうすることで，処理を追加するときに間違いが起きにくくなるためである。

ある条件を満たしていたとき，満たしていないときそれぞれについての処理を実行するときには， else を使う。

if 文で実行する文のあと else を書くことで， if に続く条件が満たされない場合に実行する処理を与えることができる。

switch 文

switch 文は変数のとる値が複数あり，それに応じた処理を実現する場合によく使われる制御文である。 ただ条件に合致する場合にある処理をするなら if 文をいくつか書けば機能的に十分であるが， switch 文ではフォールスルーという独特の技法で処理を途中から合流させるようなことができる。

switch 文は，整数リテラルか整数型の変数の値に応じた処理を行う。 文字や文字列，浮動小数点数はそのまま使うことができない。 例えば変数 a の値が 1, 2, 3 それぞれのとき case 1↵, case 2↵, case 3↵ とそれぞれ出力し， どれにも当てはまらないとき default と出力するプログラムは，

フォールスルーの処理を if 文で書く場合には，条件文がとても複雑になるため， この書き換えはあまり実用的ではない。
