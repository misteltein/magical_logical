C 言語

関数

関数とは文の集まりであり，プログラムの部品と捉えられるようなものである。 基本的に１つだけの機能をもたせることが推奨されている。 何度も使われる処理を切り出して関数にすることで， プログラム全体の見通しを確保することができる。

関数の作り方

関数が呼び出されるより前に，関数の定義をする必要があるのため， ソースコードの上部で，関数のプロトタイプ宣言を行う。 このプロトタイプとは，実装をせずに関数の名前，返す値の型，引数だけを定めるものであり，
のように書く。 返される値のことを，戻り値(return value, 返り値, 返却値)と呼ぶ。 関数に機能をもたせる実装はこの宣言文の後に書く。 main 関数よりも後に書くことが多い気がするが， プロトタイプ宣言の後ならどこでもよい。

機能を持たせるための実装は，引数を列挙した () の後に，複合文として書く。 戻り値は，return 文を使って返す。 条件に応じて返す値が異なる場合には，複数の return 文を書くことができる。

このプログラムを実行すると， main 関数が実行され，functionA が呼ばれる。 その functionA の実装にしたがって処理が進み， functionB が呼び出される。 functionB の実装にしたがって処理が進み，戻り値を戻す。 functionA がそれを使って戻り値を返す。 それは main 関数の printf の引数として使われ標準出力される。
