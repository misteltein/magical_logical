宣言文と実行文

プログラムを作成するとき， 変数や関数を作る際に宣言文，処理を書くために実行文を書く。

宣言文

宣言文とは，値を保存する領域を確保し，それに名前をつけるためのコードのこと指す。 整数を扱うための領域を確保し，そこに 123 という値を表す情報を保存し， この領域を num という名前で呼ぶことは，

実行文

実行文には，単純文・複合文・構造文が含まれる。

単純文

単純文は，関数の中で使用できる文である。 例えば

はそれぞれが単純文であり，main 関数や，自身で関数を実装するときに使うことができる。

複合文

複合文とは，{ } で宣言文や実行文を囲ってまとめたものを指す。 この内部で宣言された変数は，それが宣言されてから複合文の終わり } までの範囲で使うことができる。 この有効な範囲のことをスコープと呼ぶ。

制御文

条件に応じて処理を分岐させる if-else 文，switch 文や， 条件を満たすまでまたは一定回数の繰り返しの処理を行う while 文，do-while 文，for 文は， どれも制御文と呼ばれる。
