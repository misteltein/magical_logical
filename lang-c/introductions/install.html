C 言語

環境構築

C 言語で書かれたソースコードはコンパイルすることで計算機で実行できるようになる。このコンパイルをするために必要な環境を用意する。

環境構築の方法やインストールするパッケージのバーションなど日々変化しているため，表示例は細かく書かかなかった。ここに書いた手順をもとに，各自のコンピュータに合わせた方法を調べながら進めてほしい。

目標はコンパイラである clang をインストールすることである。

Windows 10 の場合

Ubuntu を仮想的に用意し，そこに環境を作ることができる。WSL または VirtualBox あたりを使えばできる。インストールする OS は　Ubuntu 20.04 を推奨する。Ubuntu の用意については，ここでは説明しない。また C 言語をコンパイルするための環境構築については Ubuntu の場合の手順を参考にすること。

Ubuntu を使わない場合には，llvm/llvm-project にある Assets からインストールのためのパッケージ LLVM-12.0.1-win64.exe がダウンロードできるようなので，これで用意できるかもしれない。私の手元には Windows 機がないので試していない。
