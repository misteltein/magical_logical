コンパイルするときにつけたオプション -lm は数学用のライブラリ math.h を使うときだけ必要なものだが，忘れてしまうひとが多そうなので，常につけておくことにする。

オプション Wall, Wextra, Wshadow, Wfload-equal は不適切な書き方をしているときに警告を出すためにつけている。もし今後，コンパイルの際に警告が出たら，そのメッセージをよく読み，問題の部分を修正しよう。

これらの多くのオプションを書くのは面倒なので，コマンドのエイリアスを作成することを勧める。zsh の場合には，~/.zshrc に

プログラム（ソースコード）の改変

ソースコードはテキストファイルであるため，これを書き換えるためのアプリケーションはテキストファイルをまともに扱えるアプリケーションならある程度何でも良い。UTF-8 を扱えて " " を “ ” にご丁寧に変換してくれるものでなければ問題ない。しかし，メモ帳のようなものでは何かと不便なので，Visual Studio Code や JetBrains 系統などの開発者向けのものを使うことが多い。もちろん Vim でも構わないが，無理をして使う必要はない。
