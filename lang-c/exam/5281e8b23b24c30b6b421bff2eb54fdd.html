情報科学 III 試験問題

注意事項

  • 試験中のテキストの読み直し，オンラインでの調べものを許可する
  • 試験中の学生同士のやりとりを禁止する（情報共有があれば関係者の総合点を 0 点にする）
  • 当 WEB サイト以外のコンテンツを参考にする場合，そちらのライセンスに違反すれば，総合点を 0 点にする
  • 試験問題に沿って作成したファイルは，１つのディレクトリ（フォルダ）にまとめ，学籍番号を名前とし，ZIP で圧縮し（つまり学籍番号.zipとして），Codex に提出すること
  • ファイル名はすべて半角英数で表記する
  • ファイル名が異なっていた場合には 0 点になる

    • A

    以下の要求に沿ったプログラムをそれぞれ作りなさい。プログラムの名前は A1.c のように A に続けて問いの番号を付けたファイル名にしなさい。また全体のプログラムを実行するための手順を README_A.txt という名前のファイルに他人に分かるように書きなさい。

    1. int 型の変数をつくり，その変数に 1234 という値をもたせた後，変数の値を出力する
    2. char 型の変数を 2 つ作り，それぞれの変数に何らかのアルファベットを値として与え，それらが同じならば「等しい」,異なれば「異なる」と出力する
    3. 整数 10 から 5 まで順番に 1 行ずつに分けて表示する
