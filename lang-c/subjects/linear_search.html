線型探索

配列の要素を端から順に調べ， 目的の値を検索する操作のことを線形探索という。

前提

標準出力，配列，条件分岐，反復，ファイルの読み込みのページの内容を理解していることを前提とする。 またはそれらを理解するために行うものである。

仕様

  • 配列は，テキストファイルを読み込み作成する
  • このファイルは，1 行につき 1 つの整数が書かれたもので 50 行以下である
  • ファイルの名前, 検索する値（整数）は，リテラルで与える（コマンドライン引数でも可）
  • 読み込み作成した配列について線形探索を行う
  • 目的の値が見つかればファイルの何行目であったのかを表す整数を標準出力する
  • ファイルが目的の値を含まなければ　element not found↵ という文字列を標準出力する
  • 目的の値が見つかった場合，標準出力を行った時点で処理を終了する

実行例

入力ファイルとして，入力ファイルのサンプルにある 整数 50 行, 区間=[0:50], シード=1621025592805379139 のファイルを選択したとき， 3 を検索すると，3 と標準出力され，2 を検索すると element not found↵ と標準出力される。
